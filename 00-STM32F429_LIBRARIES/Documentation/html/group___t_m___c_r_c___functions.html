uint32_t TM_CRC_Calculate16 ( uint16_t *  arr,
uint16_t  count,
uint8_t  reset 
)

Calculates 32-bit CRC value from 16-bit input array.

Parameters
*arrPointer to 16-bit data array for CRC calculation
countNumber of elements in array for calculation
resetReset CRC peripheral to 0 state before starting new calculations
  • 0: CRC unit will not be reset before new calculations will happen and will use previous data to continue
  • > 0: CRC unit is set to 0 before first calculation
Return values
32-bitCRC calculated number
