typedef tCGI TM_ETHERNET_CGI_t

Structure for CGI tags.

Note
Parameters
*pcCGINameLink, when handler function will be called, "/leds.cgi" for example
pfnCGIHandlerFunction handler, which will be called when user access to some link
Note
Handler function looks like this: const char * HandlerFunctionName(int iIndex, int iNumParams, char *pcParam[], char *pcValue[])
Function must return "URL" which will be displayed to user, for example "/index.html" to show first site
